## 기술스택 및 언어  
* JAVA 17   
* Spring 3.14
* Spring JPA ORM
* MYSQL
* IntelliJ IDEA

## 요구사항 분석 및 구현과정


| 요구사항 | 구현여부 | 참고사항(구현과정) |
| --: | :-- | :--: |
| 채용공고 등록 | 구현완료 |  |
| 채용공고 수정 | 구현완료 |  |
| 채용공고 삭제 | 구현완료 |  |
| 채용공고 전체목록조회 | 구현완료 |  |
| 채용 공고 검색기능 | 구현완료 | 추가 기능구현 완료 : (JPQL)이용해서 모든 필드명 검색 가능  |
| 채용공고 상세 조회 | 구현완료 | 추가 기능구현 완료 : 해당 회사가 올린 다른 채용공고 가 추가적으로 포함 |
| 사용자 채용공고 지원 기능| 구현완료 | 추가 기능구현 완료 : 사용자의 채용 기록들을 저장, 사용자는 1회만 지원 가능. |
